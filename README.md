> Angular 2 Example with TypeScript

## Setup
1. Run `npm i -g typescript@^1.5.0` to install TypeScript.
1. Run `npm i -g tsd` to install TypeScript Definition manager.
1. Run `tsd reinstall --save` to download typings.
1. Run `npm i -g live-server` to install live-server.
1. Run `tsc -w` to compile and watch TypeScript files.
1. Run `live-server` to run a HTTP server.
